# Money Control System (Finance Guard)

**Desktop-приложение для осознанного управления личными финансами**

Современный инструмент для контроля расходов, который помогает избежать импульсивных трат и строить финансовую подушку.

![Главный экран](screenshot1.png) <!-- добавишь свои скриншоты позже -->

## Возможности

- Несколько счетов (основной, подушка, цели, квартира и т.д.)
- Учёт доходов, расходов и переводов
- Лимиты по категориям + предупреждения (включая рисковые: казино, Steam, доставки)
- Аналитика и визуализация трат
- Учёт нестандартных графиков зарплаты
- Тёмная тема и удобный интерфейс
- Психологические insights для сознательного отношения к деньгам

## Технологии

- **Python** + **CustomTkinter** (desktop GUI)
- Локальное хранение данных (JSON / SQLite)
- Современный дизайн (Tailwind-inspired)
