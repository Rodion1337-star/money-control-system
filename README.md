# Task Manager

Современное desktop-приложение для управления задачами.

![Главное окно](screenshots/Снимок%20экрана%202026-07-06%20152421.png)

## Возможности

- Добавление, редактирование и удаление задач
- Отметка задач как выполненных
- Категории (Работа, Личное, Учёба, Другое)
- Приоритеты с цветовой индикацией
- Сроки выполнения
- Поиск и фильтры
- Сохранение всех данных в JSON файл
