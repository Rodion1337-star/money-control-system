# Task Manager

Современное приложение для управления задачами.

![Главное окно](screenshots/%D0%A1%D0%BD%D0%B8%D0%BC%D0%BE%D0%BA%20%D1%8D%D0%BA%D1%80%D0%B0%D0%BD%D0%B0%202026-07-06%20152421.png)

## Возможности

- Добавление, редактирование и удаление задач
- Отметка выполненных задач
- Категории и приоритеты
- Сроки выполнения
- Поиск и фильтры
- Сохранение в JSON
